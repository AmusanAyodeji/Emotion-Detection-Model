# Emotion Detection App

**Contents**
- `app.py` - Streamlit frontend (upload image, display result)
- `api.py` - Flask backend providing a REST endpoint for image emotion analysis
- `model.py` - Wrapper that uses DeepFace pretrained models to detect emotion from images
- `requirements.txt` - Python dependencies
- `web_address.txt` - Deployed web address

**Instructions**

- Create a virtual environment and install the dependencies in requirements.txt
- Run "python app.py" in the terminal
